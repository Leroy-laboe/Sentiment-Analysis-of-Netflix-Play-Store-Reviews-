Make sure you have **Python** installed. If not, download it from [python.org](https://www.python.org/).


Once the installation is finished, run this command in your terminal:
```bash
python -m streamlit run app.py
```

### 5. View in Browser
The terminal will provide a link (usually `http://localhost:8501`). Copy and paste it into your browser to start analyzing reviews!
